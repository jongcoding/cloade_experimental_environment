# Atlas Tech 2025 Q3 예선 (3회)

저스틴비버 CTF Labs(Atlas Tech)가 주최한 세 번째 공개 CTF 대회의 예선 아카이브입니다.

## 대회 개요

- 대회명: Atlas Tech 2025 예선 (Atlas Tech 2025 Q3 Preliminary)
- 기간: 2025년 10월 11일 10:00 ~ 10월 12일 22:00 (KST, 36시간)
- 형식: Jeopardy, 온라인
- 참가: 일반부(gen) 단독 트랙, 총 284팀 등록 / 251팀 실제 풀이 제출
- 운영 인원: 출제 10명, 인프라 4명, 운영/안내 5명
- 예산: AWS 크레딧 $9,000 규모 (클라우드 테마로 인한 리전/계정 복수 사용)
- 플래그 포맷: `atlas2025{...}`

## 테마: "Cloud Native Offense"

3회 대회는 **클라우드 보안**을 핵심 테마로 기획했습니다. 전체 4문제 중 2문제가
클라우드·AI 관련이고, 나머지도 클라우드 CI/CD 환경을 전제로 합니다.
이 방향성은 이후 2026 예선에서도 이어질 예정입니다.

## 카테고리 분포

| 카테고리 | 문제 수 | 비고 |
|---------|--------|------|
| web     | 1 | GraphQL introspection + mutation 권한 경계 |
| cloud   | 1 | S3 버전 복원 (신설) |
| ai      | 1 | RAG 경계 조건 (KB 내 기밀 유출) |
| misc    | 1 | 소프트웨어 공급망 (typosquatting) |

## 최종 순위 (상위 3팀)

1. `cumulus_0day` (1720pts, 3문제 + 1st blood × 2 + 2nd blood × 1)
2. `s3_overflow` (1600pts)
3. `prompt_smiths` (1520pts)

## 회고 요약 (내부용)

- 클라우드 테마에 대한 반응 매우 긍정적. 특히 `cloud-s3-timetravel`이 "실무
  감각을 정확히 건드린다"는 피드백.
- AI 카테고리 `ai-rag-boundary`는 2024 `ai-prompt-smuggle`의 확장 버전. KB 내부
  기밀을 모델이 누설하는 경계를 다룬 실용적 문제.
- 2026 대회를 준비하며 **"웹 + 클라우드 AI 체인"** 테마로 다음 예선을 예고함.
  출제 진영 내부 회의에서 언급된 2026 테마는 외부에 공개되지 않음.
