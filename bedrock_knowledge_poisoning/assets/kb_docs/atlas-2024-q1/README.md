# Atlas Tech 2024 Q1 예선 (2회)

저스틴비버 CTF Labs(Atlas Tech)가 주최한 두 번째 공개 CTF 대회의 예선 아카이브입니다.

## 대회 개요

- 대회명: Atlas Tech 2024 예선 (Atlas Tech 2024 Q1 Preliminary)
- 기간: 2024년 9월 7일 10:00 ~ 9월 8일 22:00 (KST, 36시간)
- 형식: Jeopardy, 온라인
- 참가: 일반부(gen) 단독 트랙, 총 207팀 등록 / 184팀 실제 풀이 제출
- 운영 인원: 출제 8명, 인프라 3명, 운영/안내 4명
- 예산: AWS 크레딧 $6,500 규모
- 플래그 포맷: `atlas2024{...}`

## 변경점 (vs 2023)

1. **AI 카테고리 신설**: 프롬프트 인젝션/모델 악용 문제를 정식 도입.
   출제자 `atlas-product-team/karen`이 Anthropic·OpenAI API 활용 가이드라인을 참고.
2. **난이도 상향**: 평균 solves 목표를 8~12팀으로 낮춤. 1회 대비 약 1.5배 난이도.
3. **인프라**: Ansible + Terraform 기반 자동 복구. 문제 서버 장애 시 평균 복구
   시간이 30분 -> 4분으로 단축.
4. **Dynamic Flag**: 일부 pwn/web 문제에 동적 플래그 도입. `anticheat` 커널 모듈
   프로토타입이 여기서 처음 도입되었으며, 2025 대회에서 안정화됨.
