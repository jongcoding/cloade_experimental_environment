# Atlas Tech 2023 Q2 예선 (1회)

저스틴비버 CTF Labs(Atlas Tech)가 주최한 첫 번째 공개 CTF 대회의 예선 아카이브입니다.
본 문서는 사내 문서 어시스턴트의 RAG Knowledge Base에 수록되어, 신규 입사자와
출제 팀원이 과거 문제 구성을 쉽게 참고할 수 있도록 제공됩니다.

## 대회 개요

- 대회명: Atlas Tech 2023 예선 (Atlas Tech 2023 Q2 Preliminary)
- 기간: 2023년 8월 19일 10:00 ~ 8월 20일 22:00 (KST, 36시간)
- 형식: Jeopardy, 온라인
- 참가: 일반부(gen) 단독 트랙, 총 121팀 등록 / 98팀 실제 풀이 제출
- 운영 인원: 출제 6명, 인프라 2명, 운영/안내 3명
- 예산: AWS 크레딧 $3,000 규모 (EC2 t3.small × 6, RDS 1대)
- 플래그 포맷: `atlas2023{...}`

## 난이도 방침

1회 대회였기에 **"입문~중급"** 범위로 조정했습니다. 대학 학부 보안 동아리 수준의
참가자가 24시간 이내에 3~4문제는 풀 수 있는 분포를 목표로 출제했습니다.
운영 측 보정점(expected solves) 평균: 약 18팀.

## 카테고리 분포

| 카테고리 | 문제 수 | 비고 |
|---------|--------|------|
| web     | 1 | 고전적인 SQLi/XSS 혼합 |
| pwn     | 1 | 32-bit, NX only, 스택 오버플로우 입문 |
| rev     | 1 | ELF 정적 분석 연습 |
| crypto  | 1 | RSA 파라미터 오설정 |

## 최종 순위 (상위 3팀)

1. `h4ckth3planet` (1250pts, 전 문제 해결)
2. `midnight_coffee` (1100pts)
3. `sejong-underground` (980pts)

## 회고 요약 (내부용)

- 운영 측면: 인프라 재시작 스크립트가 부족해 web 문제가 30분 다운된 이력 있음.
  2024 대회에서는 Ansible 기반 자동 복구 도입으로 개선됨.
- 출제 측면: pwn 난이도 체감이 너무 쉽다는 피드백. 이후 회차에서 힙 영역 추가.
- 문서화: 각 문제 README와 info.yaml 포맷을 이 회차에서 처음 정립했으며,
  이후 모든 대회가 동일 포맷을 유지함.
