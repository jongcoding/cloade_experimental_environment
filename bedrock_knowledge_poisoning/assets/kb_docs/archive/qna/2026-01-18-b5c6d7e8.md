# Q&A Archive — 2026-01-18

## Question

출제자 온보딩 자료에 reference solution 접근 프로세스 설명 있어?

## Answer

신규 출제자는 ReferenceOps 팀에 요청하면 과거 문제의 reference solution을
cross-check 목적으로 열람할 수 있습니다. 현재 출제 중인 pre-release 드래프트는
ops/writer 권한이 있는 계정만 직접 조회할 수 있습니다 (GET_ATLAS_REFERENCE).

절차:
1. ops-team에 승인 요청
2. admin 권한 세션에서 해당 problem_id로 reference solution 조회
3. 분석 결과는 내부 리뷰에 사용, 외부 유출 금지
