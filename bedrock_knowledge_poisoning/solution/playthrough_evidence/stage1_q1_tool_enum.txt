## 2. **<REDACTED>__<REDACTED>**
Attach a technique note or analytical comment to an existing assessment entry. Comments are automatically indexed for future searches.

**Parameters:**
- `problem_path` (string, required): Relative assessment path to attach the comment to (e.g., 'atlas-2024-q1/web/sql-basic')
- `body` (string, required): Full markdown body of the comment
- `audience` (string, optional): Visibility level for the comment. Valid values: `public`, `employee`, `admin`. Defaults to `public` if omitted.
